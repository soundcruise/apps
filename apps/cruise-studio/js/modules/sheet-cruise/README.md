# 譜面クルーズ（sheet-cruise）

Phase 2 で実装予定のモジュール。弾き語り教材用コード譜の編集・表示を担当する。

- 入力: 曲情報 / セクション / 小節グリッド / コード / 歌詞 / ドレミ
- 出力: StudioProject（js/core/song-model.js のスキーマ）
- 表示: A4紙面レンダリング（Phase 3）とブラウザ印刷

設計は `../../../APP_CONCEPT.md`（5章）と `../../../ROADMAP.md`（Phase 2/3）を参照。
「譜面は曲データのビュー」という中心思想を守り、見た目専用データを曲データへ入れないこと。
